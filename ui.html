📍 How Token Theme Works:

▼

1. Generate a neutral scale (50-950) from your base color

2. Scale values are mapped to semantic tokens automatically

3. Bold tokens use your key color:

• text/neutral-bold → key color

• icon/neutral-bold → key color

• border/neutral-bold → key color

• container/neutral-bold → key color

4. Other tokens use scale colors:

• surface/neutral-default → scale 75

• text/neutral-subtle → scale 600

• border/neutral-subtle → scale 200

5. Skipped layers:

• semantic/common/* tokens are preserved

• Text with Typography styles not containing body or label

• Layers with Card in name

• Layers with title or Title in name

• Layers with sectionHeader in name (case-insensitive)

• Layers with isSelected=false in name

• Layers with isDisabled=true in name

6. Select Frame(s) and click "Apply to Selected Frames"
